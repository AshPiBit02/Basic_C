  1. RAM(Random Access Memory
    • SRAM(Static Ramdom Access Memory
      1. It is made up of transistors and flip flops.
      2. For single block of memory six transistors are used.
      3. It has no charge leakage property so, does not need to be power-refreshed.
      4. Refresing circuit is not implemented.
      5. It utilizes less power.
      6. It is more expensive.
      7. It is faster than DRAM.
      8. It has low density.
      9. Use as cache memroy.
    • DRAM(Dynamic Random Access Memory
      1. It is made up of with capacitors and few transistors.
      2. For single block of memory only one tansistor is used.
      3. It has charge leakage property so, to be refreshed after each read operation.
      4. Refresing circuit is implemented.
      5. It utilizes more power.
      6. It is less expensive.
      7. It is slower than SRAM.
      8. It has high density.
      9. Use as main memroy.

  2. ROM(Read Only Memory)
    • Masked ROM
      1. The very first ROMs were hard-wired devices that contained a pre-programmed set of data or instructions.
      2. These kinds of ROMs are known as masked ROMs.
      3. The contend of such ROMs have to be specifed before chip production.
    • Programmable ROM(PROM)
      1. It is a programmable read only memory.
      2. The user can write instructions on PROM only once.
      3. The instrucitons written by the user cannot be erased form PROM.
      4. If there is an error while writing on PROM, it becomes unusable.
      5. It provides less usability as instructions are written only once.
    • Erasable Programmable ROM(EPROM)
      1. It is a erasable programmable read only memory.
      2. The user can write instructions on EPROM many times.
      3. The instrucitons written by the user can be erased form EPROM using UV rays.
      4. If there is an error while writing on EPROM, it can be used again.
      5. It provides more usability as instructions are written many times.
      6. EPROM chip need to be taken out the computer or device to erase the content.
    • Electrically Erasable Programmable ROM(EEPROM)
      1. It is electrically erasable programmable read only memory.
      2. The instructions written in EEPROM can be erased and reprogrammed using electric charge.
      3. The content is erased byte by byte.
      4. Selective programming can be done.
      5. EEPROM chip need not be taken out the computer or devices to erase the content
    • Flash ROM
      1. It also called flash BIOS or flash memory.
      2. It is a type of constantly powered non-volatile memeory that can be erased and re-programmed in blocks.
      3. Many modern PCs have their BIOS stored on a flash memory chip so that it can be easily updated if necessary
      4. BIOS is sometimes called flash BIOS.
