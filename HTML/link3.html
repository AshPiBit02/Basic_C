  1. First Generation Computer (1945-1954)

    • These computer were based on the vacuum tube technology.
    • These computer were very large and required a lot of space for installation.
    • Thousands of vacuum were used, they generated a large amount of heat. So, massive air-cooling system was required.
    • Punch card for input and printout display for output.
    • Acoustic delay lines as main memory and magnetic tape as secondary storage.
    • Operating frequency (processing speed) was few kilohertz.
    • These were non-portable adn slow in nature.
    • They lacked in versatility. i.e. only used for scientifice research not available for commercail use.
    • Since machine language eas used, these computers were difficult to program and use.
    • No programmed operating system was used.
    • mark I, ENIVAC, EDVAC, UNIVAC,etc. are example of first generation computers.

  2. Second Generation Computer (1955-1964)

    • These computers were based on transistor technology.
    • They were smaller as compared to first generation computers.
    • Better portability and generated less amount of heat as the first generation computers but still required air conditioning.
    • They relied on punched cards for input and printouts for output.
    • Magnetic core was used as primary memory and magnetic disks as secondary storage devices.
    • Computational time changed form milliseconds to microseconds.
    • More reliable and faster than first generation computers.
    • Reduced in size and cost of computer.
    • Batch operating system was used.
    • Programming was done mainly through assembly language.
    • Avaiable for commercial use as well as scientific purpose.
    • IBM 1401, ICL 2950/10,PDP-8,IBM 7090 etc. are example of computer of computer of second generation.

  3. Third Generation Computer (1965-1979)

    • These computers were based on intergrated circuit technology
    • They were portable and more reliable than reiable than the second generation computers.
    • Produced much less heat and therefore didn't require a dedicated cooling system.
    • Semiconductor memory for internal storage and magnetic disks were used for auxiliary.
    • Keyboards and monitors were introducted for input and output.
    • Computational time reduced from micro seconds to nanoseconds(around 12 MHz).
    • Started to use multiprogramming language OS like UNIX and MULTICS.
    • High level programming language were used.
    • Low cost computers were commercially manufactured.
    • IBM 360,NCR 390,PDP-11,STAR,etc. are examples of computer of third generation.

  4. Fourth Generation Computer (1980-Present)

    • These computer are microprocessor based computers.
    • High speed and large size semiconductor memory.
    • Magnetic disk, optical disk, and portable storage devices have been intorduced.
    • Fast processing speed(GHz become common).
    • More advanced GUI based operating system is used.
    • Multimedia, networking and distributed computing are introduced.
    • These are small in size, highly, reliable and portable.
    • Fourth generation computers are the cheapest among all the previous generation.
    • Consumes a very small amount of electricty.
    • Can be easily programmed using different types of programming language.
    • All different types of Input/Output capabilities available.
    • Wide ranges of sophistcated software become available.
    • CRAY-1,Apple II, IBM PCs,Intel PCs etc. are the example of the fourth generation computers.
